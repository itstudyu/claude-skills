> Analyzed on 2026-04-02

No database schema detected in source files. This project is a collection of Markdown skill definitions and Bash hook scripts with no persistent data storage.

## File-Based Data Structures

### Instincts (learned patterns)
- **Source:** `instincts/errors/*.md`, `instincts/code-patterns/*.md`, `instincts/review-patterns/*.md`
- **Schema (YAML frontmatter):**
  | Field | Type | Purpose |
  |-------|------|---------|
  | id | string | Unique identifier (e.g., `error-a1b2c3d4e5f6`) |
  | trigger | string | Pattern that activates the instinct |
  | confidence | float (0.0-1.0) | Auto-calculated: min(0.9, 0.3 + (occurrences - 1) * 0.2) |
  | domain | string | Category (e.g., "error-handling") |
  | source | string | Origin ("auto-detected" or manual) |
  | occurrences | integer | Number of times pattern observed |
  | first_seen | date string | First detection date |
  | last_seen | date string | Most recent detection date |
- **Source:** `.claude/hooks/learn-from-errors.sh:165-176`
- **Loading threshold:** confidence >= 0.8 — `.claude/hooks/load-instincts.sh:47`
